Introdução

O curso de programação de Algoritmos e Programação de Computadores não adota a utilização de IDEs durante as aulas práticas e sim o uso do terminal.

O terminal é chamado frequentemente de linha de comando ou shell. Até pouco tempo, esta era a maneira que o usuário interagia com o computador; entretanto, os usuários de Linux viram que o uso do shell pode ser mais rápido do que um método gráfico e que ainda merece algum mérito hoje. Aqui você encontrar os comandos mais usados durante o curso.

Comandos

List Command (ls)

O comando ls é usado para mostrar todos os arquivos e pastas que se encontram no respectivo diretório diretório, basta digitar ls no terminal.

FOTO USANDO LS

Change Directory (cd)

O comando cd é usado para trocar de um diretório para outro, ou seja, serve para voce conseguir entrar em uma pasta, basta digitar cd NomeDaPasta ou cd Endereço da Pasta. Vale reforçar que só é possível entrar em um diretório caso o comando digitado combinado com o atual diretório de extamente no diretório desejado.

Ou seja, caso você tenha uma pasta A e dentro dela uma pasta B e dentro da B uma pasta C, se você está na pasta A e quer ir para a C, não irá funcionar o comando cd C, pois dentro da pasta A não existe nenhuma pasta C

FOTO MOSTRANDO JEITO ERRADO

FOTO MOSTRNADO JEITO CERTO

Também é possivel voltar um diretório, para isso basta usar o comando cd ..

FOTO VOLTANDO

Move (mv)

O comando mv é usado para se mover um arquivo para um novo diretório ou pasta, exatamente igual quando arrastamos um arquivo de um lugar para o outro com o cursos do mouse.

Para se utilizar o comando mv basta digitar mv NomeDoArquivo DiretórioParaOndeVai

Make Directory (mkdir)

O comando mkdir permite que o usuário consiga criar um novo diretório, exatamente igual quando clicamos com o botão direito e depois em "nova pasta".

Para utilizar o comando basta digitar mkdir NomeDaPasta

FOTO COM MKDIR

Remove Directory (rmdir)

O comando rmdir é o contrário do comando mkdir, mesma forma de usar porém ao invés de criar uma pasta, ele a deleta

Para utilizar o comando basta digitar rmdir NomeDaPasta

FOTO COM RMDIR

Make file (touch)

O comando touch é utilizado para se criar um arquivo, exatamente do mesmo jeito que voce cria um .txt ou .doc usando a interface do seu SO

Para utilizar o comando basta digitar touch NomeDoArquivo.Extensao

FOTO COM TOUCH

Remove (rm)

O comando rm é o contrário do comando touch, mesma forma de usar porém ao invés de criar um arquivo, ele o deleta

Para utilizar o comando basta digitar rmr NomeDoArquivo.Extensao

FOTO COM RM

Clear Screen (clear)

O comando clear serve apenas para limpar a tela do terminal, pois as vezes o terminal fica com tantos comandos que a poluição visual acaba dificuldando, e com este comando ele limpa tudo.

Para utilizar o comando basta digitar clear

FOTO COM CLEAR
